RC Column Design — ACI 318 (V1.4)

Educational Tool: Slender Column Analysis + Interaction Diagram
A — Design Assumptions
\(\rho_g\) —
\(\rho_g = A_{st}/A_g\). Only for Step 0. Actual from Interaction Diagram.
Frame —
Nonsway: limit = min(34-12M1/M2, 40). Sway: limit = 22.
B — Materials
f'c MPa
Concrete strength from cylinder tests. For Ec, EI, diagram. Typical: 25–40 MPa.
fy MPa
Steel yield strength. Standard: 420 MPa (Grade 60).
C — Factored Loads
Pu kN
Factored axial \(P_u = 1.2D + 1.6L\). /Ag = Y-axis (MPa).
M1 kN·m
Smaller moment. + single / − double curvature. Affects \(C_m\).
M2 kN·m
Larger moment (always +). Min: \(M_{2,min} = P_u(15+0.03h)/1000\).
\(\beta_{dns}\) —
\(\beta_{dns} \approx 1.2D/P_u\). For EI calculation. Typical: 0.4–0.7.
\(\phi\) —
Tied: 0.65. Spiral: 0.75. \(\phi\) embedded in SI diagram curves.
D — Geometry
b mm
Column width. \(A_g = b \times h\).
h mm
Column depth in bending direction. For r, Ig, γ.
lu mm
Clear height between supports. Not story height.
k —
k=1.0 pinned, k=0.5 fixed, k>1 sway. Buckling = \(k l_u\).
d' mm
\(d' = cover + tie + d_b/2\). For \(\gamma\). Typical: 50–70 mm.
