WHAT IS THIS TOOL?
This tool designs an RC column per ACI 318 in 6 steps. Every formula is explained.
SECTION A — Design Assumptions
ρg (assumed) —
Assumed reinforcement ratio for preliminary sizing (0.01–0.015 for economy)
ρg = Ast/Ag. ACI allows 1%–8%. Start 1%–1.5% for economy. Used in Step 0 only.
Frame Type —
Nonsway = has shear walls. Sway = no walls. Affects slenderness limit.
SECTION B — Material Properties
f'cMPa
Concrete compressive strength (cylinder, 28 days)
f'c = 28-day cylinder strength. Typical: 25–40 MPa. Affects Ec, Po, and interaction diagram.
fyMPa
Steel yield strength (420 = Grade 60)
fy = yield stress of bars. Grade 60 = 420 MPa. Affects Po and interaction diagram.
SECTION C — Factored Loads
PukN
Factored axial load (e.g. 1.2D+1.6L)
Pu = factored axial compression. Column must satisfy BOTH Pu and Mu simultaneously.
M1kN·m
Smaller end moment (+single, −double curvature)
M1 = smaller end moment. +ve = single curvature (worse), −ve = double (better). Can be zero.
M2kN·m
Larger end moment (always positive)
M2 = larger end moment. SHORT: Mc=M2. SLENDER: Mc=δns×M2. Min: Pu(15+0.03h)/1000.
βdns—
Sustained load ratio ≈ 1.2D / Pu
βdns ≈ 1.2D/Pu. Higher → more creep → larger magnification. Typical: 0.4–0.7.
ϕ—
ϕ = 0.65 tied, 0.75 spiral. Reduces nominal capacity for safety.
SECTION D — Geometry & Reinforcement
bmm
Column width (short dimension)
b = short dimension. Affects Ag, ties, and Ig.
hmm
Column depth (direction of bending)
h = depth in bending direction. Affects r, Ig, γ. Increasing h greatly helps.
lumm
Unsupported length (clear height)
lu = clear height between supports. λ = k·lu/r. Typical: 2500–4000mm.
k—
Effective length factor
k = effective length factor. Nonsway: 1.0. Sway: 1.2–2.0+.
covermm
Clear cover to ties
Cover to tie outside. d' = cover + d_tie + db/2. Typical: 40mm.
dbmm
Bar diameter
n (bars)bars
Number of bars (min 4)
Ast = n×π·db²/4. ρg must be 1%–8%.
d_tiemm
Tie diameter
