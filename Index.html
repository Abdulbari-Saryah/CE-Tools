🧮 CE-Tools

Interactive Structural Engineering Design Tools
ACI 318 · Bilingual · Free & Open Source
Educational tools for reinforced concrete design per ACI 318. Each tool provides step-by-step calculations with detailed explanations — because understanding WHY matters more than just getting the answer. Built with HTML + JavaScript + MathJax. Works offline in any browser.
📂 Available Tools
🏗️
RC Flexure Design
Design rectangular RC sections for flexure per ACI 318. Covers: required steel area (As), minimum steel, tension-controlled check, bar selection, and capacity verification. Suitable for beams and one-way slabs.

RC Column Design
Design RC columns per ACI 318 with all 6 steps: gross area, steel ratio check, ties/spiral, slenderness, design strength (φPn), and demand vs capacity. Covers short and slender columns with moment magnification.
