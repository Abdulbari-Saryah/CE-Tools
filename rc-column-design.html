WHAT IS THIS TOOL?
This tool designs an RC column per ACI 318 in 6 steps. Every formula is explained.
SECTION A — Design Assumptions
\(\rho_g\) assumed —
Assumed reinforcement ratio for preliminary sizing (0.01–0.015 for economy)
\(\rho_g = A_{st}/A_g\). ACI allows 1%–8%. Start 1%–1.5% for economy. Used in Step 0 only.
Frame Type —
Nonsway = has shear walls. Sway = no walls. Affects slenderness limit.
SECTION B — Material Properties
f'c MPa
Concrete compressive strength (cylinder, 28 days)
\(f'_c\) = 28-day cylinder strength. Typical: 25–40 MPa. Affects \(E_c\), \(P_o\), and interaction diagram.
fy MPa
Steel yield strength (420 = Grade 60)
\(f_y\) = yield stress. Grade 60 = 420 MPa. Affects \(P_o\) and interaction diagram.
SECTION C — Factored Loads
Pu kN
Factored axial load (e.g. 1.2D+1.6L)
\(P_u\) = factored axial compression. Column must satisfy BOTH \(P_u\) and \(M_u\) simultaneously.
M1 kN·m
Smaller end moment (+single, −double curvature)
\(M_1\) = smaller end moment. +ve = single curvature (worse), −ve = double (better). Can be zero.
M2 kN·m
Larger end moment (always positive)
\(M_2\) = larger end moment. SHORT: \(M_c=M_2\). SLENDER: \(M_c=\delta_{ns} \times M_2\).
\(\beta_{dns}\) —
Sustained load ratio ≈ 1.2D / Pu
\(\beta_{dns} \approx 1.2D/P_u\). Higher → more creep → larger magnification. Typical: 0.4–0.7.
\(\phi\) —
\(\phi\) = 0.65 tied, 0.75 spiral. Reduces nominal capacity for safety.
SECTION D — Geometry & Reinforcement
b mm
Column width (short dimension)
b = short dimension. Affects \(A_g\), ties, and \(I_g\).
h mm
Column depth (direction of bending)
h = depth in bending direction. Affects r, \(I_g\), \(\gamma\). Increasing h greatly helps.
lu mm
Unsupported length (clear height)
\(l_u\) = clear height between supports. \(\lambda = k \cdot l_u/r\). Typical: 2500–4000mm.
k —
Effective length factor
k = effective length factor. Nonsway: 1.0. Sway: 1.2–2.0+.
cover mm
Clear cover to ties (typical 40mm)
Cover to tie outside. \(d' = cover + d_{tie} + d_b/2\). Typical: 40mm.
db mm
Longitudinal bar diameter
n (bars) bars
Number of bars (min 4)
\(A_{st} = n \times \pi d_b^2/4\). \(\rho_g\) must be 1%–8%.
d_tie mm
Tie diameter
